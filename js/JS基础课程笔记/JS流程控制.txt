流程控制有三种 顺序结构 分支结构 循环结构

顺序结构  按照代码顺序依次执行
分支结构 根据不同的条件得到不同的结果 if语句 switch语句
if （条件表达式）{执行语句} 
执行思路 如果条件表达式结果为真 则执行大括号内的执行语句； 如果条件表达式结果为假则不执行大括号内的语句 而是执行if后面的代码

if else语句
if （条件表达式）{
        执行语句一
}     else {
        执行语句二
}
执行思路 如果表达式结果为真那么执行语句一，否则执行语句二 最终只有一个语句执行 else后面直接加大括号

多分枝语句 利用多个条件选择不同的结果 多选一
if else if
if (条件表达式) {
   语句1；
} else if （条件表达式2）{
   语句2；
} else if（条件表达式3）{
   语句3；
}  else {
   最后语句；
}
