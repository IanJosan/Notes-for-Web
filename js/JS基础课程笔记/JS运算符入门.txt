运算符
operator 也被称为操作符 用于实现赋值比较和执行运算

常见运算符

算数运算符
+ -  * / %取余  4%2=0 5%3=2 3%5=3
浮点数运算不如整数  浮点数不能直接判断是否相等
判断能否整除 取余是否为0
表达式是由数字、运算符、变量组成的式子 我们成为表达式如1+1
表达式都有结果 叫做返回值 
在程序里面把右边表达式计算完把结果返回值写在左边 2=1 + 1


递增递减运算符
递增++  递减 --
++ -- 可以写在变量前面也可以写在后面，放在前面就称为前置递增/递减运算符，放在后面成为后置
递增递减运算符必须配合变量使用
前置和后置单独使用时没有区别，但是后置自增是先返回原值 后自加1


比较运算符
<  
> 
<= 
>=  
== 等于运算符 比较两者 判断是否相等 默认转换数据类型会把字符串型的数据转换为数据型的
!=  不等于号
=== 全等 要求值和数据类型完完全全一致
!==


逻辑运算符
与布尔值有关系，返回值是布尔值
逻辑与 &&  且一个是false 结果是false
逻辑或 || 都为false 结果为false
逻辑非 ！

短路运算 逻辑中断：当有多个表达式时，左边的表达式值可以确定结果时就不再继续运算右边的表达式的值
逻辑与 表达式1 && 表达式2 
如果第一个表达式的值为真，则返回表达式2
如果第一个表达式的值为假，则返回表达式1
逻辑或 的短路运算
如果表达式一的值为真，则返回表达式一 
如果表达式一的值为假，则返回表达式二


赋值运算符
= 直接赋值
+= -= 加、减一个数后再赋值 num=num+2   //num += 2
*=  /=  %= 乘除取模后再赋值


运算符的优先级
小括号》一元运算符》算数运算符》关系运算符》相等运算符》逻辑运算符》赋值运算符》逗号运算符
一元运算符 ！逻辑非比 ++ -- 优先级高
逻辑运算符 &&逻辑与比逻辑或||高
