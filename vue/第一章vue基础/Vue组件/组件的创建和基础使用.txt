每个组件都是一个独立个体，代码里体现为一个独立的vue文件

1，创建组件，封装要复用的标签样式JS为一个.vue文件
2，注册组件
全局注册，在main.js中
import Vue from 'vue'
import 组件对象 from 'vue文件路径'
Vue.component("组件名"，组件对象)

局部注册  在某个.vue文件内 常用
import 组件对象 from 'vue文件路径'
export deault {
      components：{
           “组件名”：组件对象
     }
  
}
3，使用组件，当作标签使用。把标签最终替换成组件里面封装的标签使用

----------------------------------------
注意点：Vue组件内样式只针对当前组件内标签生效如何使用？
在各个组件的style 标签上添加scoped属性。
原理：scoped会将各个组件内的所有标签添加一个相同特殊的 data-v-hash哈希值以区分来自不同组件的标签，做到不同组件标签的互不影响
