watch可以侦听data/computed属性值的改变
语法
watch:{
    "被侦听的属性名"(newVal,oldVal){

      }
}
被侦听的属性名可以是data里面的变量也可以是计算属性里面的变量

-----------------------------------------
深度侦听和立即执行
侦听复杂类型或者立即执行函数
语法：
watch:{
   "要侦听的属性名"：{
          immediate:true, //网页打开立即执行
          deep:true, //深度侦听复杂类型内变化
          handler (newVal,oldVal){

             }
      }
}
----------------------------------------
$watch 方法和watch原理一样，但是更加灵活，他不是在组件初始化就立即监听，而是可以写在函数中，函数调用时才监听
