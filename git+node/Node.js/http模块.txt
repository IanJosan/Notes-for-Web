http模块是用来创建web服务器的模块
http.createServer()
需要导入
const http = require('https')

普通电脑和服务器区别在于 服务器上安装了服务器软件 例如 apache iis 等

在node.js中不需要使用这些，可以手写代码手写服务器软件提供web服务

IP地址是每台计算机的唯一地址
通常用点分十进制表示成（a,b,c,d）的形式其中a,b,c,d都是0~255之间的十进制整数 如 IP（192.168.1.1）

互联网中每台web服务器都有自己的IP地址 可以在终端运行 ping www.baidu.com 来查询百度的IP

在开发期间，自己的电脑即是服务器也是客户端，测试可以在浏览器中输入127.0.0.1这个IP地址，就可以把自己的电脑当作服务器访问

域名和域名服务器
IP地址和域名是一一对应的关系，域名服务器就是提供IP地址和域名之间的转换服务的服务器 DNS服务器
127.0.0.1对应域名localhost

端口号
web服务器都对应一个端口号，客户端发送过来的请求，通过端口号就可以被准确的交给对应web服务端处理
每个端口号不能同时被多给web服务站用
URL中80端口可以被省略
