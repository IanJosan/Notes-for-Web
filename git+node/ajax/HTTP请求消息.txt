客户端发起的请求叫做HTTP请求
客户端发送到服务器的消息叫做HTTP请求消息 或HTTP请求报文

HTTP请求消息由 请求行 请求头部 空行 请求体 四部分组成

请求行 由 请求方式 url HTTP协议版本，他们之间用空格隔开
GET  URL  HTTP版本 

请求头部 描述客户端的基本信息
由多行键/值对 每行的键和值之间用英文冒号分割
User-Agent 说明当前是什么类型浏览器
Content-Type 描述发送到服务器的数据格式
Accept 描述客户端能接收什么类型的返回内容
Accept-Language 描述客户端期望接收那种人类语言的文本内容

空行
通知服务器请求头部到此结束 分割请求头部与请求体 

请求体
请求体中存放的是要通过post提交到服务器的数据。
只有post请求会有请求体，get请求没有请求体
